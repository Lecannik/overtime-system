# 🕒 Overtime System AI (Интеллектуальный учет переработок)

**Overtime System AI** — это инновационная экосистема для автоматизации учета сверхурочных работ, объединяющая мощь **Artificial Intelligence (ИИ)**, мессенджера **Telegram** и продвинутой **Web-аналитики**. 

Система создана для компаний, где сотрудники работают «в полях», а руководителям нужен прозрачный контроль времени, местоположения и содержания выполненных задач.

---

## 🌟 Ключевые инновации (AI & Geo)

### 🎙️ AI Voice-to-Action (Whisper + T5)
Сотруднику больше не нужно печатать текст. Он просто записывает голосовое сообщение в Telegram-боте.
*   **Whisper ASR (OpenAI):** Распознает речь в текст (оффлайн, прямо на сервере).
*   **T5 NLP Summary:** ИИ анализирует текст, исправляет ошибки и генерирует краткую **«Суть»** переработки для менеджера.
*   **Audio Player:** Руководитель может прослушать оригинал записи прямо в Веб-интерфейсе.

### 📍 Двойной гео-контроль (Anti-Fraud)
Система исключает приписки времени. Бот принудительно запрашивает геопозицию дважды:
1.  **START:** При начале работ.
2.  **FINISH:** При завершении (вместе с голосовым отчетом).
*В Веб-аналитике менеджер видит обе точки на интерактивных картах Google Maps.*

### 🧹 Автоматическая очистка (Data Hygiene)
Встроенный **Cleanup Service** каждую полночь сканирует хранилище и удаляет аудиозаписи старше **90 дней**, защищая сервер от переполнения и соблюдая политики хранения данных.

## 🚀 Развертывание (Docker)

Система полностью контейнеризирована и оптимизирована для работы с тяжелыми ИИ-моделями.

1.  Клонируйте репозиторий и настройте `.env` (см. `.env.example`).
2.  Убедитесь, что на хосте есть папка `./models/cache` для весов ИИ (автоматически скачиваются при первом запуске).
3.  Запустите всё одной командой:
    ```bash
    docker-compose up --build -d
    ```

**Адреса после запуска:**
*   **Frontend (Dashboard):** `http://localhost:8080`
*   **API (Docs):** `http://localhost:8000/docs`
*   **Telegram-бот:** Имя бота, указанное в вашем `@BotFather`.
